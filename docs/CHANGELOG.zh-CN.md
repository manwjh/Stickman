## [0.1.0] - 2026-01-17

### 🎉 首次发布

AI 火柴人故事动画生成器首次公开发布！

### ✨ 新增

#### 核心功能
- **AI 驱动的动画生成** - 使用 LLM 将自然语言故事转换为火柴人动画
- **多 LLM 提供商支持** - 通过 LiteLLM 集成 OpenAI、Anthropic 和 PerfXCloud
- **交互式 Web 界面** - 现代化响应式 UI，实时动画预览
- **SVG 动画引擎** - 使用 GSAP 实现流畅的 60 FPS 动画
- **多角色支持** - 生成最多 5 个角色的动画
- **场景管理** - 支持多场景动画和平滑过渡

#### 国际化 (i18n)
- **双语 UI** - 内置中英文支持，可实时切换语言
- **本地化文档** - 完整的中英文文档
- **i18n 框架** - 自定义轻量级国际化系统

#### 配置
- **双配置系统** - 分离系统配置和 API 密钥以提高安全性
- **基于 YAML 的配置** - 易于编辑的配置文件
- **环境变量支持** - 通过环境变量覆盖设置
- **多提供商配置** - OpenAI、Anthropic、PerfXCloud 的预配置模板

#### 开发体验
- **清晰架构** - 组织良好的代码库，职责分离清晰
- **类型验证** - 基于 Pydantic 的动画数据验证
- **完善文档** - 快速开始、API 文档、配置指南等
- **启动脚本** - macOS/Linux 和 Windows 的便捷启动脚本
- **错误处理** - 优雅的错误处理和用户友好的错误消息

#### 文档
- **README** - 中英文双语 README，包含徽章和快速开始指南
- **贡献指南** - 中英文详细的贡献指南
- **行为准则** - 贡献者公约行为准则
- **API 文档** - 完整的 REST API 参考和示例
- **配置指南** - 全面的配置文档
- **快速开始指南** - 分步设置说明
- **GitHub 模板** - Issue 和 PR 模板，便于协作

### 🔧 技术栈

- **后端**: Flask 3.0、LiteLLM、Pydantic
- **前端**: 原生 JavaScript、GSAP 3.12、SVG
- **配置**: PyYAML
- **验证**: Pydantic 模型
- **动画**: 基于 GSAP 时间轴的动画系统

### 📦 支持的 LLM 提供商

- OpenAI (GPT-4, GPT-3.5-turbo)
- Anthropic (Claude-3)
- PerfXCloud (Qwen)
- 通过 LiteLLM 兼容 100+ 提供商

### 🌍 语言

- 英语 (en)
- 简体中文 (zh-CN)

### 📝 已知限制

- 每个动画最多 5 个角色
- 每个动画最多 10 个场景
- 生成时间：3-15 秒（取决于复杂度）
- 无动画持久化（无数据库）
- 无用户认证系统
